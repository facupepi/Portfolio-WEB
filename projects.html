PROYECTOS.

APP Lista de Tareas

Aplicación web utilizando React JS enfocada en la gestión de una lista de tareas. Consiste en un sistema interactivo que permita a los usuarios añadir tareas y marcarlas como completadas o eliminarlas.

APP Gestion de Libros

Aplicacion de escritorio desarrollada en C# e implementado SQL que permite realizar un seguimiento detallado de los libros, autores e editoriales, incluyendo su registro, consulta, actualización y eliminación.
